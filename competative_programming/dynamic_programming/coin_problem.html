Coin Problem

Given a set and a target number find the minimum amount of coins which sum to . Solvable in

To solve this you create an array vals[i] which will hold the minimum amount of coins to sum to value i.
Clearly adding a coin of value c to the sum i-c gets i and uses 1 more coin so vals[i]=vals[i-c]+1.
Using vals[0] = 0, as the way to get a value of 0 is using coins, as a base case, iterate first over each possible sum and than over each coin, applying the recursive relation to acheive a solution for all numbers less than or equal to t.

int vals[t+1];
//base case
vals[0] = 0;

}
