Graph Representations

Adjacency List Representation

This is generally the most useful structure. It consists of an adjacency list for each node, meaning a list of neighboring nodes for each node. 

vector<int> adj[N]

To represent a weighted graph, store with each node the weight of the edge to the node.

vector<pair<int,int>> adj[N]

Adjacency lists allow for easily processing adjacent nodes.

for(auto u : adj[s]) {
	//process u
}

Adjacency Matrix Representation

An adjacency matrix is a two dimensional array with index , representing if there is an edge between node and . 

int adj[N][N]

To represent a weighted graph, opposed to adj[i][j] only being 0 or 1 it would be the weight of the edge.

Because of the space complexity using an adjacency matrix can be impractical for larger amounts of nodes.

Edge List

An edge list is an ordered list of all edges in a graph, normally represented by a pair of numbers.

vector<pair<int,int>> edges

To represent a weighted graph represent an edge with a triple, the last value being the weight of the edge.

vector<tuple<int,int,int>> edges
