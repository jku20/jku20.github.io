Terminology

Basic Graphs

A graph is a pair of sets. A set of vertices, sometimes called nodes and a set of two-sets (a two set is a set with two distinct elements), edges, each edge containing two vertices.

A path is a sequence of edges which leads from one vertex to another. It is called a cycle if the first vertex is the same as the last and called simple if each vertex appears at most once.

A graph is connected if there is a path between any two nodes. The connected parts of a graph are called its components. A tree is a connected graph with nodes and edges such that there is a unique path between any two nodes.

A graph is directed if edges can only be traversed in one direction. A graph is weighted if each edge is assigned a weight, often interpreted as edge lengths.

A directed graph has indegree, the number of edges going into a node, and outdegree, the number of edges going out of a node.

A coloring of a graph assigns every node a color such that no adjacent nodes have the same color. A graph is bipartite if it can be colored with only two colors.
