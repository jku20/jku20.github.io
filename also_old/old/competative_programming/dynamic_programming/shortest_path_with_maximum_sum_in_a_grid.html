Shortest Path With Maximum Sum

Given an grid with numbers at each point in the grid, starting at the top left corner going only right and down what is the path with maximum value to the bottom right. Solvable in .

Notice and . Iterating over possible and using the function gives a solution.

Implementation

int grid[n+1][n+1];
int sum[n+1][n+1];
sum[0][0] = 0;
for(int i = 1; i <= n; i++) {
	for(int j = 1; j <= n; j++) {
		sum[x][y]=max(sum[x-1][y],sum[x][y-1]) + grid[x][y];
	}
}
cout << sum[n][n];
